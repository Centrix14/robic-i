# Исток [![NO AI](https://raw.githubusercontent.com/nuxy/no-ai-badge/master/badge.svg)](https://github.com/nuxy/no-ai-badge)
Исток — это client-side веб-приложение для поддержки метода исследования соответствий и отклонений. На данный момент приложение находится в переработке.

## Соглашение о коммитах
Используется система коммит-сообщений, близкая к conventional commits. Теги:
- `fn` — метод/функция,
- `test` — тест,
- `src` — неклассифицируемый исходный код,
- `uml` — UML-диаграмма,
- `file` — файл,
- `doc` — документация,
- `misc` — всё остальное.

Перед тегами указывается характер изменения:
- `+` — дополнение, расширение;
- `-` — удаление,
- `~` — изменение.
